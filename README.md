* UT基本準則
  1. 一個測試案例只測一種方法
  2. 最小的測試單位
  3. 不與外部（包括檔案、資料庫、網路、服務、物件、類別）直接相依
  4. 不具備邏輯
  5. 測試案例之間相依性為零
